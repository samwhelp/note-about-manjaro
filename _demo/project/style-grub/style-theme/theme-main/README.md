## Usage

### install

``` sh
$ ./install.sh
```

or

``` sh
$ make install
```


### package-install

``` sh
$ ./package-install.sh
```

or

``` sh
$ make package-install
```


### config-install

``` sh
$ ./config-install.sh
```

or

``` sh
$ make config-install
```


### asset-install

``` sh
$ ./asset-install.sh
```

or

``` sh
$ make asset-install
```


### asset-remove

``` sh
$ ./asset-remove.sh
```

or

``` sh
$ make asset-remove
```
