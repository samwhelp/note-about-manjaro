# sakura 微調

* [腳本](https://github.com/samwhelp/note-about-manjaro/tree/gh-pages/_demo/adjustment/tool/sakura)


## Link

* [https://discover.manjaro.org/packages/sakura](https://discover.manjaro.org/packages/sakura)

### Install

run

``` sh
pamac install sakura
```

or run

```
pacman -S sakura
```


### Remove

run

``` sh
pamac remove sakura
```

or run

```
pacman -R sakura
```
