## sakura

https://discover.manjaro.org/applications/sakura


### Install

run

``` sh
pamac install sakura
```

or run

```
pacman -S sakura
```


### Remove

run

``` sh
pamac remove sakura
```

or run

```
pacman -R sakura
```
